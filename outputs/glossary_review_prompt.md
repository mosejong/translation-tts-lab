# 용어사전 검수 프롬프트

> 아래 프롬프트를 GPT / Gemini / Claude 등에 복사해서 사용하세요.
> 데이터는 `glossary_review_all_langs.txt` 에서 해당 언어 섹션만 잘라 붙이면 됩니다.

---

## 공통 프롬프트 (언어만 바꿔서 사용)

```
한국 어린이집·유치원·초등학교 가정통신문에서 쓰이는 용어 번역을 검수해 주세요.

검수 기준:
1. 해당 언어를 모국어로 쓰는 학부모가 일상에서 자연스럽게 이해할 수 있는가?
2. 너무 딱딱하거나 공식적인 표현은 아닌가?
3. 의미가 한국어 원문과 다르거나 어색한 번역은 없는가?
4. 고유명사(앱 이름 등)나 통화 단위(won, ₩)가 잘못 번역된 경우는 없는가?

출력 형식:
- 문제없음: OK
- 수정 필요: [한국어 원문] → 현재: [현재 번역] → 제안: [더 나은 번역] / 이유: [한 줄 설명]

아래 용어 목록을 검수해 주세요:

[여기에 glossary_review_all_langs.txt 에서 해당 언어 섹션 붙여넣기]
```

---

## 언어별 추가 지시사항

| 언어 | 추가 지시 |
|---|---|
| 베트남어 (vi) | 남부/북부 방언 차이 고려, 호치민 기준 일상 표현 선호 |
| 중국어 (zh) | 간체자, 중국 본토 학부모 기준 (대만식 표현 피할 것) |
| 태국어 (th) | 존댓말 수위 확인 (너무 격식체이면 구어체로 수정) |
| 말레이시아어 (ms) | 말레이시아 표준어 기준 (인도네시아어와 다를 수 있음) |
| 몽골어 (mn) | 키릴 문자 표기 확인 |
| 러시아어 (ru) | 학부모 대상 친근한 표현 (공문서 어투 피할 것) |
| 일본어 (ja) | 히라가나/가타카나/한자 혼용 자연스러운지 확인 |

---

## 검수 순서 추천

1. **vi** — 가장 먼저 사용될 언어, 최우선
2. **en** — 검수자 구하기 쉬움, 빠르게 확인 가능
3. **ja** — 한자 공유로 오번역 발견 쉬움
4. **zh** — 간체 표현 자연스러운지 확인
5. **th / ms / mn / ru** — 외부 검수자 필요
