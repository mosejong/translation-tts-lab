# NLLB v3 20 Google Back-Translation Review

검수 기준: NLLB 베트남어 번역 결과를 Google 번역으로 한국어 역번역하여 의미 보존 여부를 확인했습니다.

## Summary

첫 20개 결과는 그대로 서비스에 쓰기 어렵습니다. 특히 준비물, 비용, 학교 행정 용어에서 치명적인 오역이 나왔습니다.

주요 문제:

- `도화지`, `색칠 도구`가 물감/페인팅 도구로 흔들림
- `정서행동 검사`가 관계 테스트/감정 테스트처럼 번역됨
- `원복`이 셔츠로 축소 번역됨
- `체험학습`이 실험/체험수업처럼 번역됨
- `돌봄교실`이 진료실/관리실처럼 번역됨
- 금액이 사람 수로 바뀜: `40000원` -> `40,000명의 참가자`
- `앞치마`, `머릿수건`이 `무릎`으로 오역됨
- `재난대피훈련`이 난민 교육처럼 번역됨

## Decision

현재 단계에서 NLLB 기본 번역을 그대로 사용하지 않습니다.

다음 순서:

1. 용어 사전을 더 채웁니다.
2. `vietnamese` 정답 번역을 사람이 검수해서 채웁니다.
3. 사전 기반 후처리를 먼저 적용합니다.
4. 정답 데이터가 충분해지면 LoRA 파인튜닝을 검토합니다.
