# Presentation Evidence

## Core Message

일반 번역기를 그대로 쓰는 것이 아니라, 가정통신문에서 학부모가 실제로 해야 할 일을 놓치지 않도록 구조화하고 검수하는 시스템입니다.

짧은 설명:

> 파파고나 구글 번역은 문장을 번역하지만, 우리 서비스는 가정통신문에서 날짜, 금액, 제출물, 준비물을 뽑아 쉬운 문장으로 정리하고, 학교/유치원 용어 사전을 적용한 뒤 음성 안내까지 제공합니다.

## Why Not Just Papago?

일반 번역기는 편리하지만, 학교/유치원 안내문에서는 다음 문제가 생길 수 있습니다.

- 학교 행정 용어가 일관되게 번역되지 않습니다.
- 준비물이나 제출 서류가 잘못 번역되면 실제 행동 오류로 이어집니다.
- 금액이 사람 수처럼 번역되는 등 숫자 해석 오류가 생길 수 있습니다.
- 유치원 표현인 `등원`, `하원`, `원복`처럼 문화적 맥락이 필요한 단어가 흔들립니다.
- 번역 결과가 맞는지 자동으로 확인하기 어렵습니다.

## What We Add

| Component | Role |
| --- | --- |
| 쉬운 한국어 문장 | 번역 전 입력을 짧고 명확하게 정리 |
| 용어 사전 | `원복`, `체험학습`, `돌봄교실` 같은 도메인 용어 고정 |
| 오역 감지 목록 | `đầu gối`, `tị nạn`처럼 명백한 오역 자동 표시 |

| 파인튜닝 후보 데이터 | 검수된 문장쌍으로 모델 개선 실험 |
| TTS | 초급 학부모를 위한 음성 안내 |
