# MVP Demo Result

이번 샘플은 번역/TTS 파이프라인이 끝까지 성공한 케이스다.

샘플 흐름:

- 쉬운 한국어 입력을 받는다.
- NLLB 기반 베트남어 번역 결과를 생성한다.
- 학교 용어사전 기반 glossary check를 수행한다.
- Edge-TTS로 베트남어 mp3 음성을 생성한다.

이번 번역문은 전체적으로 자연스럽지만, `도시락` 같은 준비물 핵심 용어가 번역문에서 누락되었다. glossary check는 이를 `도시락 -> cơm hộp` `missing_term`으로 감지했다.

이 결과는 일반 번역 결과를 그대로 신뢰하면 안 되고, 학교 특화 용어사전 기반 검수 루프가 필요하다는 근거다. 따라서 MVP의 핵심은 단순 번역이 아니라 `번역 + 용어사전 검수 + TTS` 흐름이다.

고정 데모 산출물은 `outputs/mvp/demo_case_01/`에 저장한다.
